# SQL_Sakila
## Simple SQL project using the Sakila DB
### This script was created in MySQL Workbench and utalizes such commands as:
* ORDER BY
* GROUP BY
* Use of Wildcards %, _
* WHERE, LIKE, IN
* ALTER TABLE
* ADD COLUMN
* DROP COLUMN
* HAVING
* CREATE TABLE
* JOINS
* Subqueries
* CREATE VIEW
* DROP VIEW
